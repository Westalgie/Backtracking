Дан граф. Определите последовательность вершин, через которые необходимо пройти, чтобы нарисовать этот граф. Нельзя отрывать карандаш от бумаги и проходить по одной и той же дуге несколько раз. //Нажатие по форме в режиме рисования точек создает вершину. Последовательное нажатие на две разные точки в режиме рисования граней создает грань. Если грань не создется, то нужно нажимать на обе те же точки в обратном порядке
